# salturno-privacy

Página pública de la política de privacidad de la app **Salturno** (gestión
personal de turnos hospitalarios). Servida vía GitHub Pages a partir de
`index.md`.

La política se actualiza editando `index.md`. El historial vive en git: cada
cambio queda fechado y firmado por un commit.

## Por qué un repo aparte

El código de la app es privado. La política de privacidad necesita ser
pública y enlazable desde App Store Connect. Separarla en su propio repo
evita exponer el código fuente al cumplir el requisito de URL pública.
